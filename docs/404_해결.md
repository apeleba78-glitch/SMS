# 404 "This page could not be found" 해결

## 1. 항상 홈부터 시작하기

**올바른 시작 주소:** `http://localhost:3000` (끝에 경로 없이)

- 여기서 **층 선택** → 층 클릭 → 공간 카드 → 공간 상세 → 이슈 목록/접수 순으로 이동하세요.
- 주소창에 **직접** `/floor` 만 입력하면 404가 납니다. `/floor/층id` 형태여야 합니다.

## 2. 404가 났을 때

- 화면에 **「홈(층 선택)으로」** 버튼이 보이면 눌러서 처음부터 다시 진행하세요.
- 상단 **공간 관리**를 눌러도 홈으로 갑니다.

## 3. Vercel 등 배포 환경에서 404가 날 때

- **동적 경로** (`/floor/[id]`, `/room/[id]`, `/issue/[id]`) 는 서버가 있어야 합니다. `next build` 후 `next start` 또는 Vercel 기본 설정이면 동작합니다.
- **정적 내보내기**(`output: 'export'`)를 쓰면 동적 경로가 빈 페이지/404가 될 수 있으므로, 이 프로젝트는 **export 사용하지 않는** 방식으로 배포하세요.
